# Linear and Logistic Regression Analysis

This project implements linear and logistic regression models from scratch in Python.

- Predicts building energy efficiency using regression
- Classifies bankruptcy risk using logistic regression
- Compares analytical solutions vs gradient descent
- Explores effects of hyperparameters (learning rate, batch size)

Results:
- Similar performance between analytical and SGD regression
- High classification accuracy on bankruptcy dataset
  
## Files
- `energy efficiency and bankruptcy prediction.ipynb`: main notebook
- `data/ENB2012_data.xlsx`: energy efficiency dataset
- `data/qualitative_bankruptcy_data.txt`: qualitative bankruptcy dataset
- `Report.pdf`: project report
